To successfully run the code with coco make sure to check this link: https://github.com/numbbo/coco 
First you have the first phase which is generating experimental data using cocoex 
the files in this repository for DE with CR1 and DE with CR not 1 are the scripts to obtain experimental data
once you run these two scripts after installing coco you will obtain your results in exdata/ 
To benchamrk you simply run this script: 
cocopp.main('exdata/first_folder_name exdata/second_folder_name')

you will obtain your results in ppdata folder.
